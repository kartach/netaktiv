Thank you for using J-HotelReservation, the way to automate reservation on your site. Enable bookings online and automate the reservation process to increase the efficiency of your overall lodging facility management.

Before using please read the User manual

Key features are:
  • 5 configurable reservation steps, from booking dates to final confirmation.
  • multilingual functionality
  • responsive design
  • multiple integrated payment options
  • multiple reservation modules
  • flexibility in managing reservations
  • manage rooms, taxes, extra options, email templates, reviews
  • easy and intuitive to manage from the backend
  • create multiple offers
  • use of discounts
  • client dashboard
  • airport transfer
  • course/excursion addon
  • simple, flexible way to define room prices
  • and many more...
